# Mima Petshop

## Descripción

E-commerce de productos para mascotas desarrollado en React.

## Tecnologías

- React + Vite
- React Bootstrap
- Firebase Firestore

## Funcionalidades

- Listado dinámico de productos desde Firestore.
- Detalle de producto individual.
- Agregar al carrito, modificar cantidad, vaciar.
- Validación de stock en tiempo real.
- Registro de compras en Firestore con ID de orden.
- Navegación por categorías.
